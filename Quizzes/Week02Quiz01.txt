3. What does the following code snippet do? Briefly explain.

int x=3, y=5, z;

z = x; x = y; y = z;


First Line:
- Declares three int type variables named x, y, z

- Assigns values to x and y, 3 and 5 respectively. int z is left unassigned


Second Line:
- Assigns x value to z variable, assigns y value to x variable, and assigns z value to y variable


4. What does the following code snippet do? Briefly explain.
double x = 3.2, y = 5.8;
x = x + y;
y = x - y;
x = x - y;


 First Line: 
- Declares two double type variables named x and y
- Assigns 3.2 value to x and 5.8 value to y

Second Line:
- Assigns sum of x + y to x variable

Third Line:
- Assigns subtraction of x - y to y variable

Fourth Line:
- Assigns subtraction of x - y to x variable
